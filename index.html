    Skippy.aic is the highest-compatibility SCP Wiki scraper Discord bot that allows users to display SCP articles in easily readable embeds, along with other useful and fun features.

    [Add Skippy.aic to Discord!]

    Features/Commands


    Open Source: If you have concerns with the bots you add to your servers, you can rest easy knowing that Skippy.aic’s code is completely free to view in order to ensure privacy (no matter how poo poo the code is). If you want, you can even download all the files and host your own version of the bot. Check out Skippy.aic’s GitHub!

    SCP Embeds: Just by typing ./scp and a number, Skippy.aic can retrieve all the SCP’s information, including the Description, Object Class, Disruption Class, and Risk Class at a moment’s notice.

    Article Refinement: Skippy.aic allows for JSON file overrides, which means that bugged article embeds can be fixed easily and great articles can look even better in Discord embeds.

    Webhook Enhancement: Using the Foundation’s finest interdimensional travel technology and thaumaturgy, Skippy.aic can dynamically change it’s own name and profile picture to match the other-worldly embed content perfectly using secure and immediately-disposed-of webhooks.

    Auto-Embeds: Skippy.aic can automatically detect compatible SCP Wiki URLs and respond with that article’s embed. If you share an article with friends, Skippy.aic can help to ensure the article is readable for everyone without having to click off of Discord.
    Privacy Note: The Auto-Embed feature is optional. It’s code is completely open-source, free to read, and does not store detected messages. This feature can be disabled by going to the bot’s “Skippy.aic” role and removing the “Add Reactions” permission. This will cause the Auto-Embed function to immediately cancel and not view, reply to, or attempt to process any messages.

    Highest-Compatibility: Skippy.aic supports a vast majority of typical formats used by SCP Wiki authors, including the Anomaly Classification System, the Flops Disruption Class header, the Woed Classified Bar, and, of course, the default Object Class system.
